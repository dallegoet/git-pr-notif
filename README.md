# Git PR Notif

Be notified when you assigned to a PR, or when someone answer to you on a PR, or when someone tag you on a PR

## Install

```bash
pnpm install
```

## Setup

Create a `.env` file with the following variables:

```bash
GITHUB_TOKEN=
GITHUB_OWNER=
GITHUB_REPO=
GITHUB_USERNAME=
```

## Usage

```bash
pnpm start
```
